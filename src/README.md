## 大致方法是什么？

![](./assets/51-手写公式1.png) 


1. 找出 source和driving 的 dense correspondence     
2. 基于 dense corresponence 的 warp，称其为 impicit warping 作者认为一个CA可以完成 1 和 2.     

![](./assets/IT图2.png) 

不需要显式地提取光流，Q 和 K 的相似度描述了隐式的光流。
