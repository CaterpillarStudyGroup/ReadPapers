# Maskedmimic: Unified physics-based character control through masked motion

**文章要解决的问题：**
- 多模态输入的运动实时控制

**主要技术点：**
1. 以CVAE为生成模型，输入多模态控制信号输出控制结果
2. CVAE的输入使用掩码方式，能够兼容各种任务的输入需求，通过运动补全达到统一多模态输入的目的  
3. 轨迹优化功能蒸馏到模型中，输出的是PD控制的目标而不是动画数据
4. 为了让生成模型直接学到轨迹跟踪目标，先用轨迹跟踪算法处理mocap数据，得到PD控制目标作为CVAE的训练GT
